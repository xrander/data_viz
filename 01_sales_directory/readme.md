# Superstore Sales Dashboard — R Shiny

Dark-themed, interactive sales intelligence dashboard built with R Shiny + Plotly.

---

## Stack

| Package | Role |
|---------|------|
| `shiny` | App framework |
| `bslib` | Bootstrap 5 theming |
| `plotly` | All charts (interactive) |
| `shinyWidgets` | Enhanced inputs (pickerInput) |
| `DT` | Interactive data table |
| `dplyr` / `lubridate` | Data wrangling |
| `scales` | Number formatting |
| `tidyr` | Pivot for heatmap |

---

## Setup

### 1. Download the dataset

Go to: https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

Download `Sample - Superstore.csv` and save it as:

```
data/superstore_raw.csv
```

### 2. Install packages

```r
source("install_packages.R")
```

### 3. Prep the data

```r
source("R/prep_data.R")
```

This generates `data/superstore_clean.rds`.

### 4. Run the app

```r
shiny::runApp()
```

Or from terminal:

```bash
Rscript -e "shiny::runApp('.')"
```

---

## Dashboard Sections

| Tab | Content |
|-----|---------|
| **Overview** | KPI cards, monthly revenue + orders time series, segment donut |
| **Categories** | Top 15 sub-categories bar chart, profit margin heatmap, sales vs profit bubble |
| **Geography** | US choropleth map by state, top 10 states bar chart |
| **Trends** | YoY category lines, shipping mode area chart, discount vs profit scatter |
| **Data** | Paginated, searchable order-level table |

---

## Filters

All charts respond to:
- **Year range** slider (2014–2017)
- **Region** multi-select
- **Segment** multi-select
- **Category** multi-select

## Deploy to shinyapps.io

```r
# Install rsconnect if needed
install.packages("rsconnect")

rsconnect::setAccountInfo(
  name   = "YOUR_ACCOUNT",
  token  = "YOUR_TOKEN",
  secret = "YOUR_SECRET"
)

rsconnect::deployApp(
  appDir  = ".",
  appName = "superstore-dashboard"
)
```
